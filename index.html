// Portfolio risk calculation
class RiskAnalyzer {
    constructor(portfolio) {
        this.portfolio = portfolio;
    }
    
    calculateVaR(confidenceLevel) {
        // Value at Risk calculation
        const returns = this.getHistoricalReturns();
        const sortedReturns = returns.sort((a, b) => a - b);
        const varIndex = Math.floor((1 - confidenceLevel) * returns.length);
        return sortedReturns[varIndex];
    }
    
    getHistoricalReturns() {
        return [-0.02, 0.015, -0.01, 0.025, 0.008];
    }
}

Financial Formulas

Compound Annual Growth Rate (CAGR):

\[CAGR = \left(\frac{EV}{BV}\right)^{\frac{1}{n}} - 1\]

Net Present Value (NPV):

\[NPV = \sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t} - C_0\]

Sharpe Ratio:

\[Sharpe = \frac{R_p - R_f}{\sigma_p}\]
